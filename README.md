博客参考：[基于Neo4j的电影数据可视化](https://bubblewu.github.io/ckbarwdyc0000z5yd8tl0cjqn/)

本文主要内容为：
- 基于requests + BeautifulSoup抓取时光网电影数据；
- 基于电影数据构建电影和关系实体信息；
- 数据导入neo4j进行存储分析；
- 基于Bottle框架的对neo4j数据进行查询可视化展示。
